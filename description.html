Electro – is a clean and functional theme with vertical menu, perfect for your Electronic eCommerce store project.

The template contains 36 HTML files, 4 Header Styles and 9 Color variations.

It also contains 3 completely different Homepages with 4 headers option. Any content from each single Homepage can be transfered into another one very easily.

Features

  • Built with Bootstrap
  • SASS files supplied
  • Break-down of SASS files into components and blocks
  • PHP Files included ( not working PHP files but files separated into smaller blocks)
  • Fully responsive
  • Highly Customizable
  • RTL Compatible
  • Touch Enabled Sliders
