Electro – is a clean and functional theme with vertical menu, perfect for your Electronic eCommerce store project.

The template contains 46 HTML files, 8 Header Styles.

It also contains 7 completely different Homepages with 8 headers option. Any content from each single Homepage can be transfered into another one very easily.

Features

  • Built with Bootstrap
  • SASS files supplied
  • Break-down of SASS files into components and blocks
  • Fully responsive
  • Highly Customizable
